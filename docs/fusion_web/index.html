How to use it

1. Open http://127.0.0.1:8002/.
2. Click the GET and CGI buttons to see returned content.
3. Use the POST button to send a body to the CGI script.
4. Use the DELETE button to remove /delete-target.txt and confirm the response changes to 200 or 404 on later tries.
