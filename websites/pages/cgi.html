CGI Scripts

All scripts are in website/cgi-bin/, executed by /usr/bin/python3 via your server's CGI handler.

Echo — GET with Query String

Tests that QUERY_STRING is set correctly and that the CGI response is streamed back.

CGI POST body

Echo — POST Body via stdin

Tests CONTENT_LENGTH, CONTENT_TYPE, and that your server pipes the body to the CGI's stdin.

CGI ENV dump

Environment Variables

Calls /cgi-bin/env.py — dumps every CGI env variable your server sets. Useful for verifying GATEWAY_INTERFACE, SERVER_NAME, HTTP_* headers, etc.

CGI calculator

Calculator — Query String Parsing

Tests that /cgi-bin/calc.py receives and parses QUERY_STRING correctly.

Open Echo in iframe

Renders /cgi-bin/echo.py directly in an iframe — you can see the full HTML output from the CGI script as the browser renders it.
