HTTP Methods

Test GET, POST and DELETE — and verify that disallowed methods return 405.

GET

GET — Static File

Fetch any path from your server. Try /, /pages/about.html, or /assets/css/style.css.

POST

POST — Form Data to CGI

Sends application/x-www-form-urlencoded to /cgi-bin/echo.py. Tests that your server passes CONTENT_LENGTH and stdin to the CGI process.

—
POST

POST — Raw text/plain Body

Tests raw body delivery to CGI. CONTENT_TYPE will be text/plain.

—
DELETE

DELETE — Remove Uploaded File

Send a DELETE to /uploads/filename. Upload a file first on the Upload page.

—
405

Method Not Allowed

Send a PUT to /pages/methods.html — that location only allows GET. Server must return 405.
